# webcam_streamer

Requirements

* A web camera
* Python 3.7+
* Opencv-python
* Flask


![gif webcam_streamer.gif](/webcam_streamer.gif)


```
$ git clone https://github.com/brandon-wallace/webcam_streamer.git

$ cd webcam_streamer/
```

```
$ pipenv shell

$ pipenv install
```

```
$ python3 wsgi.py
```

Browse to http://127.0.0.1:5000
